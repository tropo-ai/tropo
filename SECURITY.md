# Security Policy

If you find a security issue, please do not open a public issue. Email hello@tropo-ai.com, or use GitHub's private security advisory for this repo. We will respond promptly.

Scope note: Tropo runs locally on your own files. The most sensitive surface is your own environment and API keys, which never live in this repo (keep them in a gitignored env file). Please report anything that could expose another user's data or keys.
