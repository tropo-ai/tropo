Declares the KB Articles category within the web-target ship-artifact graph. Canonical reference articles cited by the Explainer pages ship through this category folder. Each entry is the single source of truth for a specific Tropo primitive, surfaced at the canonical UID route `/kb/<uid>`.

## Description

The KB Articles category surfaces canonical reference depth at `tropo-ai.com/kb/<uid>`. These are the primitive-depth articles — "How the Tropo Vault Works", "How Agents Work", "How Playbooks Work", etc. — that explainer pages link to as the deeper read.

Default cleanup rules per the manifest root (4a99638d §3): strip nav-block sentinels, strip Relations table, strip substrate-only sections, rewrite UID-style links (KB sibling links → `/kb/<uid>`; article links → `/agentic-builders/uid/<uid>`; substrate-only links → plain text), preserve cited-by references routed to web URLs. Default rendering template: `KBLayout` React component.
