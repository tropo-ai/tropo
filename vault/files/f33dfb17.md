## Purpose

Step 1c-crew of the UI Foundation Refactor — first sweep in the legacy-name retirement. Replace every `var(--accent)` / `var(--bg0)` / `var(--panel)` / etc. reference in `app/crew/page.tsx` (~95 refs in inline styles) and any `.crew-*` selectors in `app/globals.css` with their `--tropo-*` equivalents. Aliases from app-deploy-1 still in place; this sweep makes the references point at the canonical names directly.

**Expected visual diff: zero.** Every legacy name aliases to the same tropo token; replacing the reference name doesn't change the resolved value.

## Scope

- `app/crew/page.tsx` — 95 `var(--legacy)` references inside style attributes, sweep to `var(--tropo-*)`
- `app/globals.css` — `.crew-*` selectors (~18) that reference legacy tokens internally; sweep to `--tropo-*`

## Out of scope

- `.crew-*` class names themselves (rename = Step 4)
- The 98 inline `style={{}}` attributes with raw hex codes (= Step 5 decomposition concern)
- `--arctic` references in crew page (2 occurrences) — pre-existing broken token per [53a7a624](53a7a624.md); leave for that future cycle

## Verification commitment

- `npx tsc --noEmit` clean
- `npm run build` clean
- `/crew` route renders without visual diff against pre-change baseline
- Diff confirms only legacy-name → tropo-name swaps (no value changes)
