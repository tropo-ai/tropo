## What shipped

Atomicity via rollback manifest — forced-fail rolls back cleanly.

Rendered project board lives at `00-tropo-nav/00-tropo-active/hello-tropo/board.md` and regenerates from the substrate on every vault rebuild.

**3. Welcome Playbook v1.1 Phase 3 amendment.** A fresh user landing in Phase 3 of the Welcome Playbook walks Hello Tropo: opening line names the worked example honestly (active run in motion); Step 1 introduces the project board as the dashboard for any Tropo project; Step 2 surfaces `00-tropo-nav/00-tropo-active/` with the today/tomorrow framing (today you'll see Hello Tropo; once you start your own work, your projects sit alongside it); Step 3 invites drill-down via Tropo T1 with all 6 workstreams named as concrete anchors.
