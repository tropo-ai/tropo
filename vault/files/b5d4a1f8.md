## Purpose

Author the content artifacts the activation produces and consumes: booth collateral, demo decks, datasheets, talk decks (if speaking), and post-event nurture content. Owned by `content-coordinator-agent`.

## Structure

```
content-workstream
├── Booth collateral (one-pagers; demo cards; product datasheets)
├── Demo content (live demo flows; demo scripts)
├── Talk decks (if speaking at the event)
├── Post-event nurture (recap blog; email follow-up sequence content)
└── Internal enablement (talking points for booth staff)
```

## Nodes

Leaf workstream; tasks generated by `content-coordinator-agent`.

## Flow Rules

Linear with parallel sub-tracks (collateral + demo + talks can run independently); converges at booth-day with all artifacts ready.

## Cold-Boot Walk-Through

`content-coordinator-agent` generates tasks against the project plan: collateral drafting; demo script authoring; talk deck creation; nurture sequence content drafting. Each artifact may surface decision capsules at messaging-lock moments.

## Known Enforcement Gaps

None at v1.35.0 minimal demonstration shape.
