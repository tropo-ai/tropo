## What ships

Designed by [sa.ui record 001](../../agents/sa/sa.ui/activation-log/001-talos-t6-record.md) (first non-test fleet dispatch — verdict `pass`). sa.ui surfaced two structural facts the raw `npm run lint` output buried:

1. **Lint inflation was 5×.** Headline "31 + 15 violations" was eslint walking 4 `.claude/worktrees/<slug>/` agent scratch copies. Canonical tree had **6 unique violations across 3 files**. Adding `.claude/worktrees/**` to eslint `globalIgnores` removes the duplicate noise — single biggest reason the post-deploy count dropped from 389 to 67.

## Visual-diff risk: bounded → zero

- `<Link>` swap is a behavioral upgrade (full-page-load → client-side route + prefetch). No visible appearance change; faster nav as a side benefit.
- `<Image>` swap retains identical computed pixel sizes at all three sites. The `tropo-thinkingBrain` class's `width: 240px; height: 240px` continues to apply (no class change); explicit width/height props on `<Image>` set the underlying `<img>` attributes; opacity/filter/animation styles unchanged. The wrapper `<span>` Next inserts is `display: inline-block` and sits inside already-flex/block-positioned parents, so no layout shift expected.
