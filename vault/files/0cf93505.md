## The Five Gates

### Gate 1 — Permit Cleared
- [ ] Household permit approved by Ridgewood County
- [ ] Utility interconnection application filed and confirmed
- [ ] If household 03, 07, or 09: fire suppression addendum accepted

**Confirmed by:** Permit Research Agent 
**Evidence:** Entry in `ridgewood/permits/status.md`

---

### Gate 2 — Vendor Confirmed
- [ ] SunPath Solar installation window locked for this household
- [ ] Equipment delivery confirmed for the scheduled window
- [ ] Installer briefed on any household-specific requirements

**Confirmed by:** Vendor Coordination Agent 
**Evidence:** `INSTALLATION CONFIRMED` event in `channels/ridgewood-pilot.md`

---

### Gate 3 — Household Ready
- [ ] Consent package signed
- [ ] RC-FS-ADD addendum signed (if applicable)
- [ ] Household has confirmed availability for installation window

**Confirmed by:** Customer Onboarding Agent 
**Evidence:** `HOUSEHOLD READY` event in `channels/ridgewood-pilot.md`

---

### Gate 4 — No Open Issues
- [ ] Coordination channel has no unresolved `ISSUE FLAGGED` events for this household
- [ ] All agents have reviewed and confirmed no outstanding blockers

**Confirmed by:** Any agent 
**Evidence:** Channel scan shows clean state for this household

---

### Gate 5 — Human Sign-Off
- [ ] Mike (or Green City's authorized principal) has reviewed the household's readiness status
- [ ] Explicit approval recorded in the ops log

**Confirmed by:** Mike 
**Evidence:** Entry in `channels/ridgewood-pilot.md`: `LAUNCH APPROVED: Household [ID]`
