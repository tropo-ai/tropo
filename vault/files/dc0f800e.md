## Purpose

Bundled D3 deliverable from the locked plan: sweep every remaining `var(--legacy)` reference across the platform code to `var(--tropo-*)`, then delete the legacy alias block in `globals.css`. After this deploy, no `var(--legacy)` reference exists anywhere in the codebase (except the documented pre-existing-broken set per inbox note 53a7a624), and the legacy alias block in `globals.css` is gone.

**Expected visual diff: zero.** Every legacy name resolves to its `--tropo-*` target via the alias chain established in app-deploy-1; swapping the reference name and then deleting the alias preserves the resolved value.

## Scope

- `app/globals.css` — sweep `var(--legacy)` refs inside `.mb-*` and `.ma-*` selector rules; delete the legacy alias block created in app-deploy-1
- `components/ChatPane.tsx`, `ArtifactPane.tsx`, `author/ConversationPanel.tsx`, `author/FileViewer.tsx` — sweep `var(--legacy)` refs

## Out of scope

- Pre-existing-broken tokens (`--arctic`, `--fg`, `--fg-muted`, `--bg`, `--glass`, `--radius`, `--font-mono`, `--surface`) — these were undefined before this deploy and remain undefined after; documented in [53a7a624](53a7a624.md)
- Class-name renames (`.mb-*` / `.ma-*` / `.crew-*` → `.tropo-*`) — D7 brand sweep
- Component or page edits beyond the var() name swap

## Verification commitment

- `npx tsc --noEmit` clean
- `npm run build` clean
- All 4 routes return HTTP 200
- `grep -rE 'var\(--(accent|circuit|amber|pitfall|bg0|bg1|panel|panel2|panel-shadow|text|muted|border|ring|r-xl|r-md|gap)\b' app/ components/ lib/` returns 0
- `globals.css` no longer contains the legacy alias block
