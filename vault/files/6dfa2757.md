## Scope

**In-scope:**

- Stage 2 — Homepage blocks 1, 2, 3, 4, 7, 8 (skipping 5 + 6 with explicit code comments at the page composition site)
- Stage 3 — Token + Tailwind-defaults verification (Stage 3 gate per critic.director: NO Tailwind default color utilities on brand surfaces; all colors reference `--tropo-*` tokens)
- Stage 4 — Build + commit + push (Mike approval gate at step-3) + Vercel cutover + live smoke

**Out-of-scope (deferred to subsequent web-deploys per roadmap e6f87d14):**

- Block 5 wizard (web-deploy-5, gated on Agent-Builder Wizard brief 3a8d92c1 clearing Argus review)
- Block 6 visual map (web-deploy-3, gated on Mike's Claude Design visuals)
