## Sweep corrections (caught during verification)

The initial sweep over-renamed in two places; both corrected before commit:
- **Tailwind margin-bottom utilities** (`mb-0`, `mb-1`, `mb-1.5`, `mb-2`, etc.) — these are Tailwind built-ins, not custom classes. Sweep matched and replaced; corrective revert restored ~31 occurrences (regex `\btropo-\d` → `mb-\d`).
- **Lib import paths** (`@/lib/tropo-crew-pm`, `@/lib/tropo-crew-types`, `@/lib/tropo-crew-agent-lifecycle`) — these resolved to actual file paths that DIDN'T get renamed; sweep changed only the imports, leaving them broken. Corrective revert restored 22 import paths.
- **API contract enum** (`"crew-approved"` is a Thread["status"] union value) — sweep renamed it to `"tropo-crew-approved"`; corrective revert restored the original.

After corrections: typecheck clean, build clean, all 4 routes 200.
