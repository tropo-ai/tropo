## Purpose

Declares the Articles category within the web-target ship-artifact graph. Every published longform piece (vault entries with `type: document`, `subtype: article`, and `published_at:` populated) ships through this category folder. Direct children are individual per-piece ship-artifact entries; the category folder governs the extraction and rendering contract for all of them.

## Description

The Articles category surfaces published longform content at `tropo-ai.com/agentic-builders/[slug]`. This is the hosted-blog surface — The Agentic Builders publication. The index page at `tropo-ai.com/agentic-builders/` lists all entries sorted by publish date.

Default cleanup rules per the manifest root (4a99638d §1): strip nav-block sentinels, strip the Relations table, strip Author's notes sections, rewrite UID-style links contextually (article links → `/agentic-builders/uid/<uid>`; KB links → `/kb/<uid>`; substrate-only links → plain text). Default rendering template: `ArticleLayout` React component.

Children declared in this folder's `children:` list are the only entries that ship under this category. Discovery is explicit — adding a new article requires a paired ship-artifact entry listed here.
